# Dart Learning Plan

Here's a structured 6-week learning plan to master Dart from beginner to intermediate level. Each week builds on the previous one.

## **Week 1: Foundations**
- Dart syntax basics (variables, data types, operators)
- Control structures (if/else, switch, loops)
- Functions and parameters

## **Week 2: Collections & OOP Basics**
- Lists, Sets, Maps
- Classes, constructors, methods
- Inheritance and polymorphism

## **Week 3: Advanced OOP**
- Abstract classes, interfaces
- Mixins
- Generics

## **Week 4: Asynchronous Programming**
- Futures and async/await
- Streams
- Error handling with try/catch

## **Week 5: Functional Programming & Utilities**
- Null safety
- Extension methods
- Common packages (http, json, etc.)

## **Week 6: Flutter Prep & Projects**
- Dart for Flutter development
- Build 3 mini-projects
- Best practices & debugging
